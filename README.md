## 📄 Documents Included

### 1️⃣ IDOR & OTP Security Analysis
A security-focused analysis of:
- Insecure Direct Object Reference (IDOR)
- OTP bypass risks
- Server-side authorization enforcement
- Trust boundary failures

- ### 2. GET_Vs_POST_Report_Systems.
-  This document explores the practical and security differences between GET and POST methods in report generation systems.
-  It covers:
-  Browser behavior during refresh
-  URL reproducibility and sharing
-  Copying URLs after GET and POST requests
-  Security risks such as IDOR and parameter tampering
-  The importance of server-side authentication and authorization

The key takeaway:
Security is not determined by the HTTP method. It must be enforced through proper backend validation and access control.
